 Netflix EDA Project

Overview
Exploratory Data Analysis (EDA) of Netflix content dataset to uncover patterns, trends, and content characteristics.

Key Insights
- Movies dominate the content library.
- Top genres include Drama, Comedy, and Action.
- Sharp increase in content additions post-2015.
- Missing values mostly in Director and Cast fields.

Files Included
- `Netflix_EDA_Project.ipynb`: Main analysis notebook
- `README.md`: Project summary
